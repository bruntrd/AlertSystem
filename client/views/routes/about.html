About the Alert System Application

  • The main goal of this application is to show how easy it is to set up an alert system in any setting
  • How it works
    • This is just a demo, but basically we are sending an alert to all the connected "users" that something has occurred
    • There are two options for sending an alert, The general sends some preset values out as an alert, and anyone connected will receive it
    • The same basic concept for the custom alert but you enter the fields yourself. The type of the alert that is sent dictates what is shown for the connected alert receivers, for now that is just a change in the color of the screen. But could be anything from sound to graphics.
    • The fields required for the custom alert are dropdown boxes.
  • Actual Application for this
    • In a real world application, it will have the options to send a custom alert as well as be connected to CAP.
